# Plasmic MCP

Control your Plasmic Studio project with Claude — add elements, take screenshots, manage tokens and CMS content through natural language.

**Full documentation:** https://app.clickup.com/9007076460/docs/8cdu53c-30698/8cdu53c-30458

**New dev onboarding — create & publish Figma designs:** https://app.clickup.com/9007076460/docs/8cdu53c-30698/8cdu53c-30598

ClickUp is the source of truth; this README is just a pointer. After `./setup.sh`, follow the onboarding guide to take a Figma/prompt/HTML design into Studio and publish it as a live Next.js site on `aihe.dev`.

---

## Quick start

```bash
git clone https://github.com/AiheApp/plasmic-mcp.git
cd plasmic-mcp
./setup.sh
```

Run `./setup.sh`, answer the prompts, restart Claude Desktop. Chrome opens automatically on first use.
